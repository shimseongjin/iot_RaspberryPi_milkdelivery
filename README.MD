# Milk Delivery
라즈베리파이를 이용하여 우유배달서비스의 약점을 보완하는 iot서비스 개발 

## 개발기간
- Iot 기초설계 (2017. 11. 30 ~ 12. 15)

## 사용 언어
- Python
- Nodejs
- Java

## 개발 툴
- Notepad++
- Android studio

## API
- Firebase

## 사용센서
- 버튼 센서
- 온습도 센서
- 버저 센서
- 초음파 센서

## 기능
- 푸시 알림
  - 센서 값이 일정 이상일 경우 푸시
 
- 안드로이드 어플리케이션 구현
  - 온도와 습도를 실시간으로 확인가능
